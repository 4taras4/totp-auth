# Introduction
TOTP based autorisation application for IOS<br>
<img src="https://github.com/4taras4/totp-auth/blob/master/itunes/Group.png" width="100%">
<br>
This application provides one time passwords to be used with the two factor authentication of Dropbox, Facebook, GitHub, Google Mail, Vk, Evenlope and many more. Make your accounts safe again!

### Features
- [x] Most popular services supports
- [x] Copying passwords into pasteboard
- [x] Scan QR codes that include accounts
- [x] Supports time-based as well as counter-based passwords
- [x] Works offline
- [x] VIPER Architecture
- [x] Widget on swiftUI
- [x] Backup data to iCloud
- [ ] Apple watch support
- [ ] Mac compability
- [ ] Test coverage
